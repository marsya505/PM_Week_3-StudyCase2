This program calculates Body Mass Index (BMI) based on the height and weight entered. Each BMI calculation will determine the weight category, such as underweight, normal, overweight, or obese, using if-else branching.

In addition to calculating, the program also stores each calculation result in a list called history in the form of map data. Therefore, all calculated results are not immediately lost, but can be saved and displayed again.

At the end, the program uses a loop to display the entire history of stored BMI calculations. From this program, we can see how the use of functions, branching, loops, and data storage in lists can work together to create a more structured and user-friendly program.
